# Секция комментариев

Виджет комментариев для B2C-сайта. Одностраничное приложение на Vue 2 без шага сборки.

## Функциональность

**Просмотр**
- Список всех комментариев с именем автора, датой и порядковым номером
- Постраничный вывод — 3 комментария на страницу
- Выбор конкретной страницы через пагинацию

**Сортировка**
- По ID или по дате
- Направление: по возрастанию / по убыванию
- Переключается нажатием на кнопку — повторное нажатие меняет направление
- Пагинация автоматически сбрасывается на первую страницу при смене сортировки

**Добавление комментария**
- Поле имени пользователя
- Текстовое поле для комментария
- Выбор даты через `vuejs-datepicker` с русской локализацией
- Валидация: все поля обязательны
- После добавления форма очищается, список обновляется без перезагрузки страницы

**Редактирование**
- Inline-редактирование в карточке комментария
- Поддерживает смену имени, текста и даты
- Отправляет `PATCH`-запрос на сервер

**Удаление**
- Удаление с подтверждением
- Список и пагинация обновляются немедленно

## API

Приложение ожидает REST API по пути `/api/comments/`.

| Метод | Endpoint | Описание |
|---|---|---|
| `GET` | `/api/comments/` | Получить все комментарии |
| `GET` | `/api/comments/:id` | Получить один комментарий |
| `POST` | `/api/comments/` | Создать комментарий |
| `PATCH` | `/api/comments/:id` | Обновить комментарий |
| `DELETE` | `/api/comments/:id` | Удалить комментарий |

Структура объекта:

```json
{
  "id": 1,
  "name": "Иван Иванов",
  "text": "Отличный отель, обязательно вернусь.",
  "date": "22.05.2025"
}
```

## Запуск

Добавить `index.html` рядом с бэкендом и открыть через браузер.

Если API находится на отдельном адресе — необходимо прописать `baseURL` в начале блока `<script>`:

```js
axios.defaults.baseURL = 'https://URL_HERE';
```

## Известные проблемы

**Сервер не подключён**

Запросы к `/api/comments/` падают с ошибкой `ERR_CONNECTION_REFUSED` — бэкенд недоступен. Для проверки виуала и функционала изначально в baseURL ссылка на файл на GH Pages. Как только будет получен адрес сервера, добавить строку:

```js
axios.defaults.baseURL = 'https://URL_HERE';
```

**Предупреждение `Failed to resolve filter: 1`**

Баг самого пакета `vuejs-datepicker` при подключении через CDN — он пытается зарегистрировать внутренний фильтр с некорректным именем. На работу приложения не влияет.
